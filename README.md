# Proje Özeti:  
Bu projede, çeşitli verisetleri ile insanların uyku kalitesi analiz edilmiştir. Uyku üzerinde aşırı spor, alkol/sigara kullanımı gibi durumların uykuyu negatif etkilediği ortaya konmuştur.
